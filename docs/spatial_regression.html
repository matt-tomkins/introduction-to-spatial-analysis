7 Spatial Regression

7.1 Introduction

Aims

The aims of this practical are to:

  1. Understand how spatial regression is different from classical regression and why it matters.
  2. Understand the differences between common forms of spatial regression.
  3. Use spatial regression to reveal relationships in a sociological context.

Application

To that end, we’ll apply non-spatial and spatial regression to predict life expectancy at the MSOA-level (Middle Layer Super Output Area) in Greater Manchester.

Data

The data for this practical are sourced from the Department of Health and Social Care (DHSC, Fingertips) and the Office for National Statistics (ONS), and are available in [data/dhsc] and [data/ons/msoa] respectively. Key attributes have been retained and are described below:

  • ONS MSOA-level geometry [Source]

    • household_poverty: the number of households (modelled) in fuel poverty.

7.2 Practical

During this practical…

[1] Load MSOA geometries and check / set map projection (British National Grid, CRS 27700)

[2] Filter to region of interest (north-west, n=932) -> Add lookup table (csv) -> Add permanent join (MSOA21CD) -> Select by Attributes = MSOA_2021_lookup_table.csv.RGN22NM = ‘North West’ -> Export Selection (this retains Join fields) -> Alternatively (keeping the features selected), remove the join, and then export -> Remove lookup table

[3] Add ONS and DHSC data (csv, n=7)

[4] Table Joins (MSOA21CD to msoa_code) - life expectancy

[5] Assess spatial autocorrelation (definitely) and look at spatial weights?

[6] Prepare for OLS (unique ID field) -> Calculate field (‘unique_id’) -> Long integer (32-bit) -> Sequential Number

[7] OLS -> Inspect output (pdf) and residuals -> Life Expectancy and Income: R2 = 0.621306

[8] Check for spatial autocorrelation in the OLS residuals -> Global Moran’s I: Clustered (z-score of 14.142665, less than 1% likelihood that this clustered pattern could be the result of random chance) -> LISA: HH clusters in Cumbria, LL in Manchester

[9] Spatial Autoregression -> SEM -> SLM

[10] Check for autocorrelation again? -> Still autocorrelated… -> Same pattern with LISA…

  • Economic Activity Status: pivoted, renamed columns, and simplified to: -> employment_dna -> employed_not_ftstudent -> employed_and_ftstudent -> economically_inactive_not_ftstudent
