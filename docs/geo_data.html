Introduction to Spatial Analysis

Chapter 1 Working with Geographic Data

1.1 Introduction

Aims

The aims of this practical are to:

  1. Develop familiarity with ArcGIS interface.
  2. Develop skills in loading and investigating vector and raster data.
  3. Understand layer symbology and the basics of map production.

Application

To achieve this, we’ll import raster (temperature and precipitation layers) and vector data (country Polygons and city Points) into ArcGIS, visualise and summarise current and future climates, and produce a map.

Data

  • WorldClim temperature and precipitation data, available in [data/world-clim] [Source]
  • Natural Earth 1:10m Cultural Vectors, available in [data/natural-earth] [Source]
  • Esri World Cities, available in [data/esri] [Source]

Tools

Zonal Statistics as Table (Spatial Analyst), Cell Statistics (Spatial Analyst Tools), Select Layer By Attribute (Data Management Tools)

1.2 Practical

During this practical…

[1] Open ArcGIS Pro and become familiar with layout

[2] Add vector data (countries + cities)

[3] Filter cities to >5 million population using Select Layer By Attribute [n=54, Export]

[4] Add example of raster data e.g., wc2.1_10m_tavg_01.tif - talk about generalisation and critical cartography e.g., Crampton and Krygier (2005)

[5] Use Cell Statistics to average all the temperature data [data/world-clim/wc2.1_10m_tavg]

[6] Repeat for precipitation [data/world-clim/wc2.1_10m_prec]

[7] Use Zonal Statistics as Table (country - temperature)

[8] Use Zonal Statistics as Table (city - temperature)

[9] Symbolise points based on temperature, population …

[10] Make a beautiful map

1.3 Extra

In the practical we’ve been working with historical climate data (1970 - 2000) from WorldClim 2, described by Fick and Hijmans (2017).

WorldClim 2 also includes future climate projections, derived from CMIP6 (Coupled Model Intercomparison Project Phase 6), a project which combines outputs from a range of global climate models (GCM) for different scenarios of societal development, known as Shared Socio-economic Pathways (SSPs).

A good overview of the SSPs is provided here.

Download the monthly average temperature datasets [tx] for 2081-2100 using the U.K. Earth System Model (UKESM1-0-LL, Sellar et al. (2019)) at a 10-minute spatial resolution here.

For SSP1-2.6 (Sustainable), SSP2-4.5 (Middle of the road), SSP3-7.0 (Regional rivalry) and SSP5-8.5 (Fossil-Fueled Development), calculate the average annual monthly temperature using Cell Statistics.

Which world cities (>5 million population) are expected to see the greatest increase in temperature by 2081 - 2100? You may want to use Zonal Statistics again…
